# butane-jekyll-theme

## Getting Started
1. `git clone https://github.com/alexcarpenter/butane-jekyll-theme.git`
2. `cd butane-jekyll-theme`
3. `rm -rf CHANGELOG.md`
4. `jekyll serve`
